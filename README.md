# StressNetV3 – Fatigue Detection System

## Overview
StressNetV3 is a real-time fatigue detection system using webcam input. It detects fatigue levels based on facial expressions, eye blink rate, and head movements.

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- EfficientNetB7

## Features
- Real-time fatigue detection
- Dual-stream architecture (spatial + temporal analysis)
- Outputs: Low, Moderate, High fatigue levels

## How it works
- Captures webcam input
- Processes facial features
- Predicts fatigue level
- Alerts user to take rest

## Dataset
- Kaggle dataset (facial fatigue images)
